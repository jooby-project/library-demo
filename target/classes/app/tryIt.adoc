An imaginary, but delightful Library API for interacting with library services and information. Built with love by https://jooby.io.

== Base URL

All requests start with: `https://library.jooby.io/library`

=== Summary

[cols="1,1,3a", grid="rows"]
|===
|Method|Path|Summary

|`+GET+`
|`+/library/books/{isbn}+`
|Get Specific Book Details

|`+GET+`
|`+/library/search+`
|Quick Search

|`+GET+`
|`+/library/books+`
|Browse Books (Paginated)

|`+POST+`
|`+/library/books+`
|Add New Book

|===

==== Response codes

* `+200+`: Success
* `+201+`: Created
* `+400+`: Bad Request
* `+404+`: Not Found
* `+500+`: Server Error




== Library

Outlines the available actions in the Library System API. The system is designed to allow users to search for books, view details, and manage the library inventory.




=== 1. Get Specific Book Details

View the full information for a single specific book using its unique ISBN.

*Endpoint:* `GET /library/books/{isbn}`



.Path Parameters
[cols="1,1,3", options="header"]
|===
|Name|Type|Description
|`+isbn+`
|`+string+`
|The unique ID from the URL (e.g., /books/978-3-16-148410-0)

|===









.Response (200 OK)
[source, json]
----
{
  "isbn" : "9791563993625",
  "title" : "No Country for Old Men",
  "publicationDate" : "1978-10-05",
  "text" : "ulobsojunfhqxjiwmpezpmttceptdmbiiwmteixghnqdyjxpbnusexipygitqes",
  "type" : "TEXTBOOK",
  "publisher" : {
    "id" : "12",
    "name" : "Maldives"
  },
  "authors" : [ {
    "ssn" : "414-07-4152",
    "name" : "France",
    "address" : {
      "street" : "5678 Verdie Pike",
      "city" : "Normandhaven",
      "zip" : "50351"
    }
  } ]
}
----


.Try it
[source, bash]
----
curl -X GET '/library/books/{isbn}'
----



=== 2. Quick Search

Find books by a partial title (e.g., searching "Harry" finds "Harry Potter").

*Endpoint:* `GET /library/search`






.Query Parameters
[cols="1,1,3", options="header"]
|===
|Name|Type|Description
|`+q+`
|`+string+`
|The word or phrase to search for.

|===






.Response (200 OK)
[source, json]
----
[ {
  "isbn" : "9791563993625",
  "title" : "No Country for Old Men",
  "publicationDate" : "1978-10-05",
  "text" : "ulobsojunfhqxjiwmpezpmttceptdmbiiwmteixghnqdyjxpbnusexipygitqes",
  "type" : "TEXTBOOK",
  "publisher" : {
    "id" : "12",
    "name" : "Maldives"
  },
  "authors" : [ {
    "ssn" : "414-07-4152",
    "name" : "France",
    "address" : {
      "street" : "5678 Verdie Pike",
      "city" : "Normandhaven",
      "zip" : "50351"
    }
  } ]
} ]
----


.Try it
[source, bash]
----
curl -X GET '/library/search?q=string'
----



=== 3. Browse Books (Paginated)

Look up a specific book title where there might be many editions or copies, splitting the results into manageable pages.

*Endpoint:* `GET /library/books`






.Query Parameters
[cols="1,1,3", options="header"]
|===
|Name|Type|Description
|`+title+`
|`+string+`
|The exact book title to filter by.

|`+page+`
|`+int32+`
|Which page number to load (defaults to 1).

|`+size+`
|`+int32+`
|How many books to show per page (defaults to 20).

|===






.Response (200 OK)
[source, json]
----
{
  "content" : [ {
    "isbn" : "9790968268734",
    "title" : "The Millstone",
    "publicationDate" : "1993-09-18",
    "text" : "kvvnqooibvzuuxridgxswbhcdztapesxgycbrntmkqfzlkkdty",
    "type" : "BIOGRAPHY",
    "publisher" : {
      "id" : "84",
      "name" : "Slovenia"
    },
    "authors" : [ {
      "ssn" : "646-62-9582",
      "name" : "Mauritania",
      "address" : {
        "street" : "62488 Maynard Pines",
        "city" : "Carsonmouth",
        "zip" : "66402"
      }
    } ]
  } ],
  "numberOfElements" : "Hafnium",
  "totalElements" : "Beryllium",
  "totalPages" : "27",
  "pageRequest" : {
    "page" : "42",
    "size" : "small"
  },
  "nextPageRequest" : { },
  "previousPageRequest" : { }
}
----


.Try it
[source, bash]
----
curl -X GET '/library/books?title=string&page=int32&size=int32'
----
